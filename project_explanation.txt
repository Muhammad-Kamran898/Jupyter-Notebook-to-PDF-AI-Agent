================================================================================
1. PROJECT OVERVIEW
================================================================================

This project, named "Jupyter-Notebook-to-PDF-AI-Agent", is a Python-based tool
that transforms Jupyter Notebooks (.ipynb files) into professionally formatted
PDF and TXT reports. What makes this agent unique is its integration with local
Ollama AI to provide automatic explanations and clarifications for code cells.

Key Features:
-------------
- Parses raw Jupyter Notebook JSON format
- Generates a Table of Contents (TOC) from markdown headers
- Uses Pygments for syntax highlighting of code
- Integrates with local Ollama AI for code explanations
- Smart auto-model selection based on notebook complexity
- Parallel batch processing for faster AI queries
- Generates both PDF and TXT output files
- Handles various cell outputs (text, HTML, images, errors)
- Renders professional PDFs with page numbers via WeasyPrint
- Retry logic for failed AI explanations
- Progress tracking with tqdm
